Instalação do Emulador

Para instalar e utilizar o Emulador duckstation, siga os seguintes passos:

  1. Acesse o site do emulador duckstation.
  2. Selecione a opção que se adequa ao usuário.
  3. Após o download, extraia o arquivo no diretório desejado.
  4. Dentro da pasta do emulador, crie uma nova pasta chamada "bios".
  5. Siga as instruções do instalador e defina um lugar para armazenar os jogos baixados.
  6. Após a instalação, é necessário baixar a ISO do jogo, podendo ser obtida em https://vimm.net/vault/5976.

BIOS PS1

Para jogar o Gran Turismo 1, é necessário ter a BIOS do PS1. Siga os seguintes passos para instalar a BIOS:

  1. Crie uma pasta "bios" no diretório do duckstation.
  2. Baixe a Bios do PS1 com esse link: BIOS
  3. Extraia o arquivo SCPH1001.BIN dentro na pasta "bios".
  4. Prossiga com o processo de instalação.
